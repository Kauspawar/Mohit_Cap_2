# 🔩 SteelSense — Defect Intelligence System

SteelSense is an AI-powered web application for detecting surface defects in steel plates using an XGBoost classification model.
It takes 27 industrial sensor inputs and predicts defect types such as scratches, bumps, and stains.
The system provides probability distributions, confidence scores, and risk-level insights for decision support.
Interactive visualizations like feature importance and gauge charts enhance interpretability and usability.
Built with Streamlit, the app is designed for real-time deployment in manufacturing quality control environments.
